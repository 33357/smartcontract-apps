# Uniswap_v2

## 应用简介

Uniswap_v2 是目前市面上应用最广泛、锁仓资金量最大的去中心化交易所协议。它不仅引领了去中心化交易所的风潮，而且为整个去中心化金融的发展奠定了一块基石。
